# Smartly.Academy Roadmap

## 🚀 Upcoming Modules

### 1. Idea Gathering App ("Next Level" Todo List)
- **Concept:** A dedicated space for capturing, organizing, and refining creative ideas ("Idea Storage" > "Task List").
- **Architecture:** Local-First (LocalStorage + JSON Export).
- **Features:**
    - Quick capture interface (mobile-friendly).
    - Categorization/Tagging (e.g., "Business", "Art", "Tech").
    - "Seed" storage for fleshing out raw thoughts into actionable projects.
    - **Seed Ideas:**
        - *DNA Trash Tracking:* Public waste (dog poop, cigarette butts) tracking via DNA analysis. Markets: Municipalities & Condo Corps.
- **Status:** In Progress (MVP Building)
- **Future:** AI Coach integration for brainstorming assistance.

### 2. Chess & Piano Coaching (AI Layer)
- **Chess:** Interactive analysis board.
- **Piano:** Web Audio API + Tone.js integration for rhythm and ear training.
- **Status:** In Design Phase

## 📝 Backlog
- User Accounts/Auth (for saving progress across devices).
- Multiplayer support for Chess/Piano.
